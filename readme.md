
# Sprint 6 — Автотесты для сервиса «Яндекс.Самокат»

## Описание проекта

Реализован набор UI-автотестов для учебного сервиса **«Яндекс.Самокат»**.  
Тесты написаны на языке **Python** и запускаются в браузере **Mozilla Firefox**.

---

## Используемые технологии

- Python 3
- Selenium WebDriver
- pytest
- Allure Framework
- Page Object Model (POM)

---

## Реализованные тестовые сценарии

### 1. Блок «Вопросы о важном» (FAQ)

- Проверка, что при клике на каждый вопрос отображается соответствующий текст ответа.
- Для каждого вопроса используется отдельный параметризованный тест.

---

### 2. Заказ самоката (позитивный сценарий)

Проверяется полный пользовательский флоу оформления заказа с двумя наборами данных и двумя точками входа:

- Кнопка «Заказать» вверху страницы
- Кнопка «Заказать» внизу страницы

Сценарий включает:
- Переход к форме заказа
- Заполнение первой и второй формы
- Подтверждение заказа
- Проверку появления модального окна об успешном оформлении заказа

---

### 3. Проверка функциональности логотипов

- **Логотип «Самокат»**  
  Проверяется, что при клике происходит переход на главную страницу сервиса.

- **Логотип «Яндекс»**  
  Проверяется, что при клике открывается новая вкладка браузера, открывается сайт **dzen.ru**.

---

## Запуск тестов

### Установка зависимостей

```bash
pip install -r requirements.txt
````

### Запуск всех тестов

```bash
pytest -v
```

---

## Allure-отчёт

### Генерация результатов

```bash
pytest -v --alluredir=allure-results
```

### Просмотр отчёта

```bash
allure serve allure-results
```

Отчёт содержит:

* Названия тестов
* Описания сценариев
* Пошаговое выполнение тестов
* Статус выполнения

---

## Особенности проекта

* Используется **Page Object Model**
* Все ожидания реализованы через **явные ожидания (WebDriverWait)**
* Отсутствуют `sleep`, `print` и неявные ожидания
* Тесты независимы друг от друга
* Используется параметризация тестов
* Поддерживается генерация Allure-отчётов
